# Diluciones & Goteo · UTI

Calculadora clínica para enfermería de terapia intensiva: factor goteo (macro/microgotero) y diluciones de drogas con referencia de dosis máxima.

## Desarrollo local

```
npm install
npm run dev
```

## Build de producción

```
npm run build
```

Genera la carpeta `dist/` lista para desplegar en Vercel, Netlify o cualquier hosting estático.
